# Real-Time Voice Pipeline

```python
# whisper_live + streaming
while True:
    audio_chunk = mic.read()
    text = whisper_stream.transcribe(audio_chunk)
    if text.endswith(".") or text.endswith("?"):
        send_to_vla(text)
```

```text
[SYSTEM] You are a helpful home assistant humanoid.
[USER] Please tidy the table and bring me a water bottle.
```
