# Add Real-Time Collision Objects

```python
from moveit_msgs.msg import CollisionObject
from shape_msgs.msg import SolidPrimitive

co = CollisionObject()
co.header.frame_id = "world"
co.id = "table"
box = SolidPrimitive()
box.type = SolidPrimitive.BOX
box.dimensions = [1.0, 1.0, 0.8]
co.primitives = [box]
co.primitive_poses = [Pose(position=Point(x=0.8, y=0.0, z=0.4))]
co.operation = CollisionObject.ADD
```
