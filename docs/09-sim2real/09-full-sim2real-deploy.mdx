# From Training to Real Robot in 3 Commands

```bash
# 1. Train (already done)
# 2. Export policy
python export_policy.py --checkpoint rdt1b_final.pt --onnx

# 3. Deploy to real Unitree G1
ros2 launch sim2real deploy_to_real.launch.py \
    policy:=rdt1b_final.onnx \
    robot_ip:=192.168.1.100 \
    enable_dr:=false # no need anymore!
```

Watch your real humanoid execute a 7-step task trained 100 % in simulation — first try.

**Full source:** `src/sim2real/deploy_to_real/`
