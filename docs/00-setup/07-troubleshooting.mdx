# Troubleshooting

| Symptom | Fix |
|----------------------------------|-----|
| `libcuda.so` not found | Reboot after driver install |
| Isaac Sim crashes on launch | `export OMNI_DISABLE_ASYNC_RENDERING=1` |
| ROS 2 bridge missing | `pip install --upgrade omni.isaac.ros2_bridge` |
| Black screen in Isaac Sim | Disable Wayland → use X11 |
| Out of VRAM | Close Chrome, reduce viewport resolution |

Full FAQ with 50+ entries available in `/docs/troubleshooting-full.md`
