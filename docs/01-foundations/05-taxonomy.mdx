# Taxonomy – From Narrow to General

| Level | Description | 2025 Examples |
|-------------|---------------------------------------------|-----------------------------------|
| Narrow | Single task, fixed environment | Amazon Sparrow, Boston Dynamics Stretch |
| Embodied | Multi-skill, structured environments | Figure 01, Digit (2023) |
| Physical AI | Foundation model backbone, language interface| OpenVLA, RT-2-X, HELIOS |
| General-Purpose Humanoid | Any task a human can do in <4 hours | Figure 02, Tesla Optimus Gen 3 (late 2025) |

We are officially in the **Physical AI → General-Purpose** transition.
