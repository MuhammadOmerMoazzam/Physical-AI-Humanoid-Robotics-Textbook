# Where You Can Legally Deploy Today

| Region       | Classification          | Allowed Without Permit | Notes |
|--------------|-------------------------|------------------------|-------|
| EU           | High-risk AI system     | Only with CE marking   | EU AI Act enforced |
| USA          | General-purpose robot   | Yes (state-level)      | Executive Order 14110 |
| China        | Class III robot         | Only in designated zones | Requires NMPA approval |
| Singapore    | Collaborative robot     | Yes                    | Most permissive |

**Recommendation:** Deploy first in Singapore → collect safety data → expand.
